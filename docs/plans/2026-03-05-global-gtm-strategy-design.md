## Identity Shift

**Old positioning:** "A USDC donation app on Solana"
**New positioning:** "The operating system for global generosity"

Glimpse is infrastructure that makes giving as simple, transparent, and borderless as sending a text. Any person, any country, any cause, any time. Zero fees. On-chain proof. Direct connection to impact.

**Tagline:** "Give. See the proof. Start a conversation."

**One-liner for judges:** "Toly said it: giving money away is a great product. We built it — and it works on mainnet today."

## The Leaky Bucket Thesis

Every nonprofit in the world is a leaky bucket. Money lost to overhead, middlemen, inefficiency, opacity. Glimpse replaces the plumbing — stablecoin rails, on-chain receipts, direct connection between giver and recipient. No leaks. Every dollar accounted for.

The universal promise: you can't think of a population that isn't being served, because everything is on Glimpse. Correct incentives produce fair outcomes. The operators who deploy dollars most efficiently win.

## Global Hub Model

### How Glimpse Scales Without Traditional Offices

Each "hub" = one Glimpse Creator with a Seeker device in a city.

```
                    Glimpse (PBC)
                    Media + Tech + Token
                         |
            +-----------+-----------+
            |           |           |
       Americas      Europe      APAC        Other
       ~37K devices  ~32K devices ~24K devices ~23K devices
            |           |           |           |
       [Creator]    [Creator]   [Creator]   [Creator]
       Lagos        Berlin      Jakarta     Dubai
       Sao Paulo    Amsterdam   Tokyo       Nairobi
```

### What Glimpse Creators Do

- Stream and document real impact (video, photos, updates)
- Create on-chain listings for local causes and nonprofits
- Post proof-of-impact content to the Glimpse feed
- Earn base pay (stablecoins) + viral bonuses + $GLIMPSE tokens
- Become the local face of Glimpse in their city

### Why This Works for Seeker

- Each Creator is a daily active Seeker user (stickiness metric)
- Content drives donations drives content (flywheel)
- Local knowledge + global rails = causes that resonate in each market
- Solana Mobile gets a showcase use case: "Seeker is how this person in Lagos documents clean water projects"

### Phase Rollout

1. **Now → June:** One city (Muscatine). Prove the loop. Win the hackathon. Close first brand partnerships.
2. **June:** 2-5 more cities. Funded by raise. First Glimpse Creators deployed to highest-density Seeker regions.
3. **Each raise funds the next wave.** Token governance decides which regions expand next. Growth is purposeful, not speculative.

## Barstool Media Model for Giving

Glimpse is the media company. Creators are employees, not volunteers.

- Base pay in stablecoins via Seeker
- Viral content bonuses
- $GLIMPSE token distribution tied to company success
- Content drives donations. Donations fund creators. Flywheel.

## Token Alignment

Only donors govern. The people spending dollars get the voice. The operators who deploy dollars most efficiently win. $GLIMPSE holders vote on capital allocation. True alignment between givers and the platform.
